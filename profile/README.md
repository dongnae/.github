# 동래고등학교 (Dongnae High School)
동래고등학교 재학 중에 교내/교외 대회에 참가하여 만든 작품들과 교내 수강 신청 프로그램과 같은 개인 프로젝트를 보존하고 공개하기 위한 저장소입니다.
